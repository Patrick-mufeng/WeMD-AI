5.17 Code Block · 三种变体

VARIANT A

完整代码文件 + 语法高亮

带 macOS 风格三色圆点顶部栏 + 语言标签 + One Dark 风格语法高亮。适合展示完整代码片段。

python

import requests import json # 调用 WeMD AI 排版 API resp = requests.post( "http://localhost:5000/api/generate", headers={"Authorization": "Bearer sk-xxx"}, json={ "article": "你的文章内容...", "theme": "cyber" } ) html = resp.json()["html"] print(f"生成成功: {len(html)} 字符")

VARIANT B

行内代码

在正文中嵌入代码片段。使用 pip install flask 安装依赖， 然后设置环境变量 DEEPSEEK_API_KEY。 调用 /api/generate 端点即可生成排版 HTML。 返回的 html 字段可直接粘贴到微信编辑器。

VARIANT C

终端命令 · Catppuccin 暗色

纯暗色终端风格，适合展示 Shell 命令、配置文件、日志输出等。不带动画圆点，更简洁。

$ terminal — 快速开始

# 克隆项目

git clone https://github.com/Patrick-mufeng/WeMD-AI.git

cd WeMD-AI

# 设置 Key 并启动（无需安装 Python）

set DEEPSEEK_API_KEY=sk-your-key

portable\start_portable.bat

# 然后直接在浏览器中调用 API

✓ Server running at http://localhost:5000

VARIANT D

亮色简洁 · 单行/短代码

浅灰底色 + 深色等宽文字。适合简短的单行命令、配置项或 API 端点路径，不喧宾夺主。

$ pip install flask requests

POST /api/generate → 返回公众号兼容 HTML 片段

DEEPSEEK_API_KEY = "sk-xxxxxxxxxxxxxxxxxxxxxxxx"

小结：四种场景对应四种变体

A · 完整代码文件 — 红黄绿圆点 + 语言标签 + 语法高亮，适合展示完整代码
B · 行内代码 — 灰色胶囊底，适合正文中嵌入 variable 或 函数名
C · 暗色终端 — Catppuccin 配色，适合 Shell 命令 / 日志 / 配置
D · 亮色简洁 — 浅灰底 + 深色字，适合单行短代码，不破坏阅读流
